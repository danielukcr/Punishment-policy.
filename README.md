    <h1>Punishment Policy</h1>

    <div class="policy-text">
        <p>
        To address recent situations where a staff member issues a sanction for an incident that occurred days earlier,
        the following rule now applies: if more than <strong>4 hours</strong> have passed, action can only be taken
        if there is an <strong>official investigation supported by sufficient evidence</strong>.
        </p>

        <p>
        If a member or staff is placed under investigation, evidence must be shared with them within
        <strong>30 minutes</strong>. Failure to provide the evidence within that time will
        <strong>void the investigation</strong> due to insufficient proof.
        </p>

        <p class="note">
        This policy is in place to support the well-being of both the community and its staff.
        </p>
